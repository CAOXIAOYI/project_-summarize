# 常用正则验证与匹配

### 1.手机号验证

```
/^(1[3-8][0-9])\d{8}$/

```
### 2.邮箱验证
```
/^([a-zA-Z0-9_-])+@([a-zA-Z0-9_-])+(.[a-zA-Z0-9_-])+/ 

```
### 3.手机号344格式
```
"12345678901".replace(/(^\d{3}|\d{4}\B)/g,"$1 ") //"123 4567 8901”

"12345678901".replace(/(^\d{3}|\d{4}\B)/g,"$1-") //"123-4567-8901”
```
### 4.价格格式
```
'99893'.replace(/\B(?=(?:\d{3})+$)/g, ',')  //'99,893'
```
